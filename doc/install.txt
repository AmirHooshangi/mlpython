Installation Instructions
=========================

Getting MLPython
----------------

Installing MLPython
-------------------

* ``hg clone whatever`` or put links to tar balls
* set up MLPYTHON_DATASET_REPO

Required libraries
------------------

BLAS, LAPACK, Numpy, Matplotlib (misc.visualize), Sphinx (documentation)

List of third-party software interfaced by MLPython

* LIBSVM
